# PIC 8x8 LED Matrix Controller

## Overview
Firmware and hardware implementation of an 8x8 LED matrix display using PIC16F877A.

## My Contribution
- Performed hardware assembly and soldering of the PCB implementation
- Assisted in circuit debugging and validation
- Verified hardware functionality with firmware integration

## Features
- Multiplexed LED control
- Real-time display output

## Hardware Used
- PIC16F877A
- MAX7219
- LED Matrix
