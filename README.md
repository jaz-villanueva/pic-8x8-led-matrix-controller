# PIC Ping Pong Game (8x8 LED Matrix)

## Overview
This project implements a two-player ping pong game using a PIC16F877A microcontroller and an 8x8 LED matrix driven by the MAX7219. The system displays the ball and paddles on the matrix, with button inputs controlling paddle movement.

The focus of this implementation is the **physical hardware realization**, including PCB assembly, soldering, and validation of a working embedded system.

## Features
- Two-player ping pong gameplay
- 8x8 LED matrix display output
- MAX7219-driven multiplexing
- Button-controlled paddle movement
- Real-time ball movement and collision detection

---

## Hardware Used
- PIC16F877A Microcontroller
- MAX7219 LED Driver
- 8x8 LED Matrix Display
- Push Buttons (Player Controls)
- Resistors and Capacitors
- Crystal Oscillator
- 5V Power Supply

---
